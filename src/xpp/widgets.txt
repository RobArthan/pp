# format: full.widget.name	WidgetClassName
# N.b., it is conventional to use the application class name "Xpp"
# not the application name "xpp" in resource files
# Use a "*" as a wildcard to match zero or more components of a resource name
# Use a "?" as a wildcard for exactly one component of a resource name.
# E.g.,
# "Xpp*copy.foreground: red" makes the copy item in all the edit menus (popups and menubar) red
# "Xpp*journal*copy.foreground: red" makes the copy item in the journal popup edit menu only red
# "Xpp*popup-edit-menu.copy.foreground: red" makes the copy item in all the popup edit menus red
# "Xpp.frame.menubar.popup_file.popup_reopen.reopen.?.foreground: red" makes the items in the reopen menu red
#
# Motif separator and sash widgets are not listed.
#
# To make this list:
#
# 1) Compile xpp with USERCFLAGS=-DLISTWIDGETS. Now run an xpp session thus:
#
#	xpp 2>t -f "" -c sh
#
# This will bring up the start-up file selection dialogue.
# Open a read-only file. yes_no dialogue: answer yes.
# Bring up options tool, choose "Execute prompts for missing newlines", apply, dismiss.
# Select a word and do execute selection: yes_no_cancel dialogue: answer cancel
# Press delete. yes_no dialogue: answer no.
# Try to open a file which doesn't exist: ok dialogue: answer ok.
# Bring up search & replace, goto line, palette, template and command line tools
# from tools menu.
# Press Help in the command line tool.
# Quit. yes_no dialogue: answer yes.
#
# 2) Run:
#	sed -e 's/ /./' < t |  sort -t. +1 | sed -e /XmSeparator/d | sed -e /XmSash/d | sed -e 's/\./ /' | \
#	awk '{print $2, $1;}' | uniq >w.txt
#
# 3) Replace non-comment parts of widgets.txt with w.txt.
#
